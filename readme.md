# Project require

- node [Version LTS](https://nodejs.dev/)

# Install project

```
$ git clone --branch  clear-code-server https://github.com/nmthangdn2000/flight-order.git
```

# Setup project

```
$ npm install
```

# Run project

- Run dev

```
$ npm run dev
```

- Start project

```
$ npm run build
$ npm start
```

- Url project [`http://localhost:3000`](http://localhost:3000)
